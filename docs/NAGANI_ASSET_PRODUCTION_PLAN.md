## 2. Final Production Direction

Nagani assets should feel:

- Premium Myanmar traditional
- Red dragon identity
- Dark lacquer atmosphere
- 24k gold ornament
- Jade accent details
- Warm fire glow
- Mobile-first readable
- Real polished business/product quality

Avoid:

- Slot reels
- Slot cabinet style
- Buffalo clone direction
- Emoji final assets
- Cheap cartoon/browser-game style
- Flat SaaS card style

---

## 3. Important Technical Decision

For ၃၆ ကောင်ထီ:

AI should NOT generate exact 36 cells.

Correct production method:

- AI asset = premium board frame / background
- Frontend code = exact 6×6 clickable number grid

Reason:

- AI image generation cannot reliably count exact cells
- Frontend grid is cleaner, clickable, responsive, and exact
