# Nagani Asset Style Brief v0.2

## Product Identity

Nagani — Premium Myanmar Traditional Games Platform

Main games:

1. ၆ ကောင်ဂျင်
2. ၃၆ ကောင်ထီ

Nagani must feel like a real Myanmar traditional gaming platform, not a cheap browser game, not a generic casino clone, and not the old slot prototype.

---

# 1. Final Visual Feeling

The final visual feeling should be:

- Premium
- Local Myanmar traditional
- Red dragon identity
- Dark lacquer game-house atmosphere
- Gold and jade accents
- Mobile-first casino-style clarity
- Trustworthy wallet and result presentation
- Alive with results, winners, and game activity

When users open Nagani, they should feel:

> “This is a real Myanmar traditional game platform.”

---

# 2. Main Color Direction

Use:

- Deep obsidian black
- Dark red lacquer
- Dragon red glow
- 24k gold highlight
- Jade green status accents
- Warm fire glow
- Soft shadow and glass depth

Avoid:

- Bright cartoon colors
- Blue SaaS dashboard feeling
- Flat gray template UI
- Random neon colors
- Western slot-machine style

---

# 3. Art Style Direction

Recommended style:

- Premium 2D illustration
- Clean animal silhouettes
- Strong readable shapes
- High-resolution
- Same lighting direction
- Same outline/frame style
- Traditional Myanmar influence
- Mobile-readable at small size

Avoid:

- Child cartoon look
- Emoji style
- Cheap clipart
- Mixed 3D and 2D art
- AI images with inconsistent faces/details
- Too much detail that becomes blurry on phone
- Western slot/casino asset style

# 6. Immersive 2.5D Game Room Direction

Nagani should eventually feel like the player has entered a real Myanmar traditional game room, not just a flat web page.

The final game screens should use a 2.5D layered approach:

- Background room layer
- Table / board foreground layer
- Dice / result animation layer
- Selected bet glow layer
- Host / dealer character layer if budget allows
- Sound and voice layer
- Result reveal animation layer

This does not require full 3D at first. A high-quality 2.5D room can feel premium while staying realistic for a web project.

---

# 7. ၆ ကောင်ဂျင် Game Room Asset Direction

Final ၆ ကောင်ဂျင် should feel like a premium red-dragon Myanmar animal dice room.

Needed later:

- Game room background
- Traditional table or dice tray
- Six animal betting area
- 6 animal icons
- Dice faces for all 6 animals
- Dice roll animation or sprite/video sequence
- Result reveal frame
- Selected animal glow
- Win/loss result glow
- Optional host/dealer character behind the table

The dice should feel physical and alive, not just text on cards.

---

# 8. ၃၆ ကောင်ထီ Game Room Asset Direction

Final ၃၆ ကောင်ထီ should feel like a premium draw room or traditional number board.

Needed later:

- Draw room background
- 36 animal/number board
- Number tile normal state
- Number tile selected state
- Winning number reveal effect
- Bet ticket visual layer
- Draw open/closed visual state
- Result published visual state

The 36 board should feel like a real draw table, not plain numbered boxes.

---

# 9. Sound and Voice Direction

Nagani should later include sound feedback.

Recommended sounds:

- Button click
- Bet confirm
- Dice rolling
- Dice stop
- Result reveal
- Win sound
- Lose/neutral result sound
- Draw result publish sound
- Optional ambient room sound

Optional voice lines:

- Bet confirmed
- Dice result
- Win result
- No match
- Draw closed
- Winning number announced

Voice should feel premium and professional, not loud or cheap.

---

# 10. Host / Dealer Character Direction

A host/dealer character can be added later, but only if the visual quality is high.

Direction:

- Friendly professional game host
- Premium local game-room assistant
- Brand-safe
- Not overly sexualized
- Consistent with Myanmar traditional atmosphere
- Optional idle animation
- Optional speaking/result animation

Do not add a low-quality character just to copy casino websites. If the character is not premium, skip it first.

---

# 11. Animation Direction

Use motion to make the game feel alive.

Recommended animation:

- Dice rolling in tray
- Dice reveal bounce or stop
- Selected animal glow
- Winning highlight
- Result reveal flash
- Button press feedback
- Live winner subtle movement
- Draw number reveal

Avoid:

- Too much shaking
- Cheap arcade effects
- Random neon explosions
- Heavy animations that slow mobile phones

---

# 12. Mobile Performance Rules

Nagani must stay smooth on mobile.

Rules:

- Use WebP for most final assets
- Use PNG only when transparency is needed
- Avoid very large uncompressed images
- Optimize background size
- Keep animation lightweight
- Test on real Android phone
- No laggy full-screen effects
- No broken image icons
- No layout jump when assets load

---

# 13. Asset Approval Checklist

Before full asset production, Gary must approve a small test set.

Approve only if the sample assets feel:

- Premium
- Myanmar traditional
- Red dragon branded
- Mobile readable
- Consistent style
- Not childish
- Not emoji-like
- Not slot-machine style
- Not cheap AI mixed style
- Good enough to build the final Nagani identity

If the test assets fail, fix style first before creating all 36 animals or full room assets.
